# Map render with BSP (Процедурная генерация с помощью BSP)

## Setup (Установка)

npm install

## Run (Запуск)

**npm run start** or (или) **node app.js**

Map 80 x 40 blocks (Карта 80 x 40 блоков)
white block - wall (белый блок - стена)
grey block - floor (сервый блок - пол)
